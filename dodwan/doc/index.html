DoDWAN -- Document Dissemination in Wireless Ad hoc Networks

DoDWAN is an opportunistic networking middleware system that allows mobile devices to exchange information using the publish/subscribe model.
Although it has designed to be used mostly on mobile devices, providing disruption-tolerant communication in challenged conditions, it can also be used on fixed devices such as desktop workstations.
